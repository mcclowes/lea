# Jest Testing Skill

This skill provides guidance for writing Jest tests, with patterns specific to interpreters, parsers, and async code.

## When to Use

- Writing unit tests for the lexer, parser, or interpreter
- Testing async operations and promises
- Setting up mocks for built-in functions
- Creating snapshot tests for formatting

## Key Topics

- Arrange-Act-Assert pattern
- Testing interpreter output
- Parser AST validation
- Async test patterns
- Jest configuration for TypeScript
