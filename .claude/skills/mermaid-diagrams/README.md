# Mermaid Diagrams Skill

This skill provides guidance for creating Mermaid diagrams, including flowcharts, sequence diagrams, and AST visualization.

## When to Use

- Visualizing AST structures
- Creating architecture diagrams
- Documenting execution flow
- Generating diagrams from code

## Key Topics

- Flowchart syntax for AST visualization
- Sequence diagrams for execution flow
- State diagrams for lexer states
- Class diagrams for type hierarchies
- HTML embedding
