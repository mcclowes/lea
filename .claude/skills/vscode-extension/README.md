# VSCode Extension Development Skill

This skill provides guidance for developing VSCode extensions, including language support, LSP integration, and packaging.

## When to Use

- Modifying the vscode-lea extension
- Adding new language features (snippets, commands)
- Integrating with the Language Server Protocol
- Packaging and publishing to the marketplace

## Key Topics

- Language configuration (brackets, comments)
- TextMate grammar integration
- LSP client setup
- Snippet authoring
- Extension packaging with vsce
