# Language Server Protocol Implementation Skill

This skill provides guidance for implementing LSP features like completions, hover, diagnostics, and navigation.

## When to Use

- Adding new LSP features to the Lea language server
- Implementing code completions for builtins
- Adding hover documentation
- Providing diagnostics for syntax errors
- Implementing go-to-definition

## Key Topics

- Server initialization and capabilities
- Completion providers
- Hover information
- Diagnostic reporting
- Definition and reference providers
