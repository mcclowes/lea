# npm Publishing Skill

This skill provides guidance for publishing npm packages, including package.json configuration, versioning, and provenance.

## When to Use

- Preparing a new release of lea-lang
- Configuring package.json exports
- Setting up semantic versioning
- Publishing with provenance

## Key Topics

- Package.json configuration
- Exports field for ESM/CJS
- Semantic versioning
- Publishing workflow
- Files to include/exclude

- `references/scopes.md` - Scoped packages
- `references/tags.md` - Dist tags (latest, beta, next)
