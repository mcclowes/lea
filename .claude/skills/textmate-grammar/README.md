# TextMate Grammar Authoring Skill

This skill provides guidance for authoring TextMate grammars for syntax highlighting in editors.

## When to Use

- Adding new syntax highlighting rules for Lea
- Fixing tokenization issues
- Supporting new language features in the grammar
- Testing syntax highlighting

## Key Topics

- Scope naming conventions
- Match and begin/end patterns
- Capture groups
- Repository organization
- Testing with VSCode inspector
