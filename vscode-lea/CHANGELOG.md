# Changelog

## [0.1.0] - 2024-12-08

### Added
- Initial release
- Syntax highlighting for Lea language
- Support for keywords: `let`, `mut`, `await`, `context`, `provide`
- Pipe operator (`/>`) and arrow (`->`) highlighting
- Early return (`<-`) operator support
- Decorator highlighting (`#log`, `#memo`, `#time`, `#retry`, `#async`, etc.)
- Context attachment highlighting (`@Context`)
- Builtin function recognition
- String, number, and boolean literal highlighting
- Comment support (`--`)
- Auto-closing brackets and quotes
- Indentation rules for function bodies
